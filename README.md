# RabbitMQ.AspNetCore
<p>
  Simple RabbitMQ Usage In Asp.NetCore Using MassTransit
</p>

## Run 
1. Install and start docker
2. Build and run using `docker-compose`
```powershell
PM> docker-compose -f .\docker-compose.yml -f .\docker-compose.override.yml up -d 
```
